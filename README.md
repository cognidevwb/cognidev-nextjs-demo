# cognidev-nextjs-demo

A fast-moving Next.js 15 demo app. It depends on two sibling CogniDev repos via
`github:` shorthand — so opening this repo in the Workbench populates:

- **Repo** tab — open PRs on this repo.
- **Org** tab — open PRs on the linked dependency repos
  (`cognidev-ui-kit`, `cognidev-api-client`).

An hourly GitHub Action keeps all three repos moving (fresh PRs + heartbeat commits).
